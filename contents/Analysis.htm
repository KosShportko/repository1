The Analysis step serves the purpose of comparing patient models to statistically obtained ideal deviations to plan the treatment accordingly.

 

The Questionnaires option is used for a custom analysis and implementation of the scoring systems. You can download scoring systems (e.g., ABO Discrepancy Index, KIG, PAR Index, etc.) from Download Center on the Settings page (see the chapter Settings).

 

Note: Make sure to set planes and segment teeth (see the chapter Preparation) and measure overbite/overjet before working with scoring systems. ABO Discrepancy Index and KIG require loading a CBCT scan and aligning it with the model to be able to perform the analysis (see the chapter Input and Alignment of CBCT/CT Scans). The KIG scoring system requires a panoramic radiography scan of the patient.

 

The Overbite/Overjet option allows you to measure the overbite and overjet distances.

 

The Dental arch analysis option provides a computation of tooth widths based on the actions performed using options in the Preparation tab (see the chapter Preparation) and compares it with the customer-defined ideal arch shape.

 

The Analysis reports option provides three officially recognized analyses: Tooth width analysis, Bolton analysis and Space analysis.

 

 

The Generate Report option organizes the data contained in both the Preparation and Analysis sections into a separate printable document.

 

The Occlusion map option enables the visualization of occlusal distances in the model set. Color scale is displayed on the right of the session window.
