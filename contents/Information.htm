An Information component allows you to give additional design directions to the users in the template, which might not be associated with a component. One example of use is "open bite in the virtual articulator" placed at the start of a modeling process.

 

To create this component while working with your appliance, follow the procedure described below.

 

Click the button under the part name in the components tab, or right-click the desired model and choose Create "Information". Set the component's properties similarly as for a shell.

 

In the features tab, click and then click Add "Information step". Add the necessary information to make sure that this step appears in the template.

 

Language to edit - allows to choose the language for this step.

 

Step name - indicates the name of the information step.

 

Action - allows to select the action that will be automatically performed on this Information step.

 

When finished, click OK.
