Once a patient is added to the system, in the patient browser, create a new model set for the selected patient either by clicking or by clicking to open the Edit Patient form and selecting . The Create Model Set form opens.

 

Note: The Create Model Set form in Patient Browser differs depending on the chosen scan type and contains additional scanning-specific options. See Technical Documentation for details.

 

Fill in the fields in the form, import/make the required scans, and click . The model set is created.

 

 

A model set can have one of the following statuses:

 

Status

 Description
Empty A model set has been created but no scans have been added yet.
Scanned The required scans either have been made/imported into a model set.
Prepared A model has been prepared or a virtual base has been created on it.
Designed Some teeth have been moved either in Virtual Setup or in the Clear Aligner workflow or a treatment proposal has been sent to the Treatment Review.
Design files The Virtual Setup Export is completed.
Production files The additions to the models are done in Mold Adjustment.
