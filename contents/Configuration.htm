The Preparation step is used to prepare the upper and/or lower model(s) for future manipulations and analysis.

 

The Setup occlusion plane step allows you to place the occlusion plane that is used at the Analysis step for 2D measurements with the Cross-section tool (see the chapter Cross-Section).

 

The aim of the Standard Planes step is to specify the local origin (origo) for the maxillary model of the virtual cast. These predefined reference planes - the horizontal, sagittal and anterior-posterior - can be used as basis for the 2D cross-section analysis.

 

Note: The occlusion and standard planes need to be predefined to be available as default position for the 2D cross-section analysis (see the chapter Cross-Section). The occlusion plane is also required for the calculation of the tooth long axis at the preparation stage.

 

Note: If the models have been prepared to include a virtual base (see the chapter Preparation of Orthodontic Models), the occlusion and sagittal planes are already defined on the model set and this step can be omitted.

 

Note: All planes can be moved and saved as additional planes in the 2D cross-section toolbox (see the chapter Cross-Section).

 

For new cases and cases for which segmentation has not been done yet, the Segmentation of maxillary model step starts with auto-segmentation.

 

When the auto-segmentation process is completed and the segmentation is done, you can manually adjust the autoplaced positions by returning back to the previous sub-steps.

 

Note that auto-segmentation cannot be performed, when:

  • your 3Shape Communicate account is not set up
  • there is no Internet access
  • the auto-segmentation service is unavailable

 

Note: Auto-segmentation does not distinguish between primary and secondary teeth, therefore, when working with children be aware of this. Ensure to validate the output.

Auto-segmentation and Export in the Background

 

The Segmentation of maxillary model step has four main objectives:

  1. It lets you define the composition of the maxillary tooth row, including missing teeth to calculate the required space. You can also set up the width measurements for missing or non-erupted teeth.
  2. It segments the teeth according to their definition so you can perform a virtual setup.
  3. It defines the axis of the crown, which will be used as basis for all tooth movements at the virtual setup stage.
  4. It allows to edit position of the rotation center of the tooth.

 

The Segmentation of mandibular model step defines the composition and segmentation of the tooth row of the mandibular model. The workflow is similar to the described for the segmentation of maxillary model.
