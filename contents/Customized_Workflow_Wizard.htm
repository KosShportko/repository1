You can create a number of custom workflows with predefined operational steps. This cleans up the software user interface and organizes steps and operations for your specific purposes.

 

Option

Description

Analyzer – starts a custom workflow that contains the following default steps: Analysis objects, Overbite/Overjet, Dental arch analysis, Analysis reports and Generate Report.

Preparation – starts a custom workflow that contains the following default steps: Alignment with CT Scan, Occlusion, Alignment for open model set, Standard Planes, Setup occlusion plane, Preparation of maxillary model and Preparation of mandibular model.

 

Note: The customized workflow wizard lets you create or edit your own workflows in order to adapt Ortho Analyzer to your particular needs. The wizard is set up on the Settings page by going to Settings > System Settings > Workflows.
