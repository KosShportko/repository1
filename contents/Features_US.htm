The Features are the specific design parameters of the components, available under the corresponding buttons in the function panel. There are also additional features (e.g., Drill Compensation) that can be applied to an existing component from the right-click menu on a model:

 

 

 

 

 

Add "Modify model"/Add Sculpt

 

The Modify model is a sculpting toolbox that allows you to make local changes to the selected component or model’s geometry.

 

The Wax knife  tool allows you to add or remove surface on the selected area. Use the Add  button in the Wax knife settings section to add material to the surface. Use the Remove  button to delete extra material or the Smooth  button to smooth the area.

 

Adjust the Radius, Level and Desired thickness parameters of the marking circle by dragging the corresponding sliders.

 

You can also use the already customized sets of parameters under a number. Activate the desired set by clicking its number. To change a set, adjust the parameters and click the Save  button. Select the Show target area checkbox to display the marking circle.

The Morphing  tool allows you to curve the surface and force in and out. The black mesh circle indicates the curve action of the tool on the surfaces parallel to the view while the blue mesh circle indicates the force in and out action on the edges and angled surfaces. The green outline indicates the radius of the area to be affected by the morphing process. You can set the radius by dragging the Radius slider.

The Plane cut  tool allows you to cut off parts of the appliance using a plane. Draw a line by holding down the left mouse button and dragging across the surface to create the cutting plane. The arrow indicates the cutting direction. To reverse the cutting direction, click Swap cut direction.

 

You can select the already created plane by choosing it from the Default planes drop-down list.

 

You can pan the cutting plane by dragging the blue control point or by setting the parameter in the Offset edit box. You can tilt the cutting plane by dragging the red control points.

 

It is possible to adjust the size of the cutting plane by selecting the Local plane cut checkbox. Set the appropriate parameter in the Smooth radius edit box to smooth the edges of the cut.

 

If selected, the Show cut-off part option shows the section that will be cut off.

 

Click the Apply  button to apply the cut.

 

Click Clear to remove the cutting plane.

The Spline cut  tool allows you to cut off parts of the appliance using a spline. Select the desired area with the spline. The area to be cut off becomes translucent.

 

It is possible to Save parameters of this tool or Load the already existing ones for another appliance.

 

Select the Spline Type configuration and set the Fillet radius parameter in the corresponding edit box. Use the Reverse Direction button to swap the selected area to be cut off. Select the Additional plane cut checkbox to create an additional plane and set the Plane offset parameter in the corresponding edit box.

 

Click the Apply  button to apply the cut.

 

Click Clear to remove the cutting spline.

Use the Global transformations  tool to stretch or shrink the appliance to the desired length and height, as well as rotate and tilt it.

 

Use the green control point to adjust the length and height of the appliance. Hold down the left mouse button and drag the control point in the desired direction. You will see the ruler to view the numerical information on the movements you perform. It is possible to rotate and tilt the appliance using the red control points.

The Remove Artifacts  tool is used to clean the surface from any artefacts. The radius of the marking circle can be customized by dragging the corresponding slider. Clicking the surface marks the selected area in red. To delete the selection, click the Clear Selection button. To remove the selected artefact, click the Apply button. The Show target area checkbox turns on/off the visualization of the marking circle.

The Split models  tool allows you to remove or keep non-attached (floating in space) parts of the appliance. Select the part(s) you want to keep or remove - they become outlined in black when selected - and enable the corresponding option. Click the Apply button to perform the action.

The Reduce thickness  tool allows you to reduce thickness on the surface by placing a spline. Place the spline by clicking along the surface. The spline will create a groove on the surface. Use the Radius edit box to indicate the radius of the groove. Make sure to respect the settings recommended by the combination of production equipment and material chosen.

 

Select the Smooth edges after cut option to smooth the edges of the resulted groove. Select the Spline closed option to create a closed spline.

 

Click the Apply button to perform the action.

 

You can Undo/Redo the operation by clicking the appropriate buttons. It is possible to view the history of actions performed by clicking the black arrow by each button.

 

It is possible to observe the changes you perform in colors. Activate the Difference map by clicking the appropriate button. Open the drop-down menu by clicking the black arrow to view the legend and customize its view.

 

Add "Attachments"

 

Attachments are 3D models that can be added to or subtracted from the surface of any model, or simply used as visual guides for the output models or appliances.

 

Note: Attachments should be added to an attachment group on the Settings page by going to Settings > Appliances > Attachments before they become available in Appliance Designer.

 

Note: Attachments properties can be edited on the Settings page by going to Settings > Appliances > Attachments. For example, it is possible to lock the scaling parameters so the attachment’s geometry cannot be changed during the design process. This is particularly useful when attachments are standard physical parts that will be mounted in the designed appliance.

 

Add "ID tag"

 

The ID Tags allow you to add easy identification to any output model for easy sorting in the manufacturing equipment.

 

It is possible to adjust the ID Tag in the desired position with the help of the blue and purple control points on the joystick.

 

Add "Drill Compensation"

 

The drill compensation ensures that the drill used in CNC milling manufacturing process is able to access the sharpest areas of the designed part. It needs to be adjusted with relation to the actual drilling tool which will be applied.

 

Drill compensation will only be available in the relevant step of the workflow if it is allowed by the given machine and material combination.

 

Note: Drill compensation is only relevant for CNC milling and should be omitted or skipped for any rapid prototyping manufacturing process.

 

Drill radius - indicates the radius at which the drill will be performed (max. tool size is 1 mm (maximum 0.6 radius)). Make sure to set the size according to the tool you have loaded in the milling machine.

 

Show area for drill compensation - marks the area which will be affected by the drill compensation.

 

Skip this feature - allows you to omit this step.
