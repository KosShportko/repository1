After selecting an unprepared model in the patient browser, click the Preparation -> Custom base icon to start the preparation of the model.

 

Note: If a model set you want to prepare contains a validated bracket setup, in order to be able to change the relative position of the jaws, or the occlusion/sagittal plane, you will have to remove the bracket validation status before preparing the model set, otherwise the preparation will start from the Virtual base step.

 

Note: If you want to redo preparation of a model set created in the Ortho System 2013 version and earlier, you will be warned that all existing data attached to this model set (segmentation, virtual setups, analyses, etc.) will be deleted and have to be redone in Ortho Analyzer if you choose to proceed.

 

In the Setup plane alignment  step, place landmarks, following the on-screen instructions, to set the occlusion plane. Toggle the visibility of the maxilla and mandible using the corresponding visualization sliders in the upper right corner. Draw a line along the center of the hard palate, on the occlusion plane, to create the sagittal plane.

 

Adjust the position of the sagittal plane manually using the control points, if needed.

 

Adjust the spline manually by dragging the control points with the cursor to the required position. From the right-click menu of the control point, you can use the Add/Remove point, Clear spline or Fast edit spline option to edit the spline.

 

Click Next to go to the mandible and place the cut spline as with the maxilla.

 

When the cut splines are set on both maxilla and mandible, click Next to proceed to fitting the base. Adjust the orientation of the base using the purple control points, and the base size using the green points of the bounding box. Click and hold the right mouse button to drag the bounding box to the required position.

 

In case of successful base creation, your models will appear to have virtual bases. Otherwise, you'll be prompted to make required adjustments. When the base is created, click Next to proceed to the next step.

 

Adjust your result using the tools available at the Sculpt Maxillary  and Mandibular  steps.

 

The Wax knife  tool allows you to add/remove/smooth material in the selected areas. The Morphing  tool allows you to stretch the target area inwards and outwards.

 

The Remove Artifacts  tool is used to clean the model from the artefacts that might be present after scanning. The target area is marked with a blue circle and can be customized by dragging the corresponding the slider. Clicking on the surface will paint the selected area red. To delete the selection, click the Clear Selection button. To remove the selected artefact, click the Apply button.

 

The Plane cut  tool allows you to cut off parts of the model using a plane. Draw a line in the desired place to create a plane. You can move the plane using the blue control point and tilt the plane using the red control points. Set the appropriate Smooth radius amount to smooth the transition between the model's edge and the cutline. To change the direction in which the plane is applied, click the Swap cut direction button.

 

The Spline cut  tool allows you to cut off some areas using a spline.

 

Note: Once the procedure has been finished, the model icon in the patient browser will indicate that the model has been prepared, that is there will be no red exclamation mark on it.

 

The Fixture  tool allows to place fixtures at the bottom side of the upper and lower jaw models.

 

Note: The Fixture option is an add-on to Ortho Planner and requires a specific license activation. Please, contact your 3Shape reseller to activate additional modules.
