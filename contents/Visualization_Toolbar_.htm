The tools in the visualization toolbar are presented in the table below:

 

Tool Description

Save session – saves the current analysis session to the default file and location.

Export models as – saves the current analysis session to a selected location.

The view tools allow you to switch between a number of predefined viewpoints:

  • Front view
  • Rear view
  • Right side view
  • Left side view
  • Top view
  • Bottom view

Real scale view – displays a model on a screen in its real size. A special scale that appears on the screen helps to manually set the model in real size by comparing an actual ruler with the scale on a screen.

/

The mouth view tools allow you to switch between the two types of cast visualization: Open mouth view and Closed mouth view.

Teeth segmentation mode – displays the teeth objects in the slider section so that you can view and manipulate them individually.

Multi view – displays six views of the same virtual cast in separate windows below: Right side view, Left side view, Rear view, Front view, Mandibular view and Maxillary view.

Cross-section – allows to create the 2D cross-section of the model at any stage of the workflow (see the chapter Cross-Section).

3D measurements and custom analysis – allows to perform 3D measurements on the model and make a custom analysis (see the chapter 3D Measurements and Custom Analysis).

Articulator (Ortho Planner only) – allows to imitate the movements of an articulator with the designed model on the screen (see the chapter Articulator (Ortho Planner Only)).

Capture screenshot – allows you to capture screenshots with a single click and save them in the patient's model set.

Axis locked rotation – allows free rotation or locks the rotation to the X, Y or Z axes.

Select one of the available ways of right-click interaction:

  • Rotate view – when selected, right-click and drag to rotate the view.
  • Pan view – when selected, right-click and drag to pan the view.
  • Zoom view – when selected, right-click and drag to zoom in and out.

Change visualization settings – allows you to adjust the objects' visualization by selecting/deselecting the available options:

  • Show scanner acquired texture – displays the texture after scanning.
  • Show material – marks the cast segments with the corresponding material colors.
  • Show shadow – toggles on/off the visualization of shadows.
  • Turn on anti aliasing – applies visual smoothing to the model's surface so that it looks less "edgy".
  • Show wireframe – shows the triangulation of the virtual casts.
