Part is an individual element of an appliance. A part consists of components and features and is typically manufactured individually. Using parts allows you to create a file that can be exported separately. For example, one appliance can be made of two separate parts: one for the upper jaw and one for the lower jaw which will be mounted together.

 

Component is a building block of an appliance. The main component types are:

  • Shell - a wrap surface with a fixed offset distance:

  • Bar - a 3D model based on a 2D profile allowing to apply predefined parameters and geometries:

  • Combine models - adding or subtracting geometries of two models.
  • Remove undercuts - fills undercuts to simulate e.g. waxing on models.
  • Model Hollowing - allows to hollow out a 3D model in order to save material for rapid prototyping.
  • Bracket transfer master model - allows you to create a master model for either thermoforming or continuous digital printing of directly printed trays. With this feature, it is possible to design an element for transferring brackets into the patient's mouth.
  • Post Processing - allows to ensure that some selected features are being applied to a specific model in a workflow.

 

Workflow is a sequence of components and features used to create an appliance, which corresponds to the chosen appliance type. All workflows are provided by 3Shape and consider the chosen production equipment and material. It is not possible for the user to create their own workflows.
