Before starting a session, a patient database needs to be created. To add a new patient, in the patient browser, click to open the Add Patient form.

 

 

Fill in the fields in the form, where the Patient ID field is mandatory, and either click  to save the patient and close the form or click to save the patient and open the Create Model Set form.

 

From the patient browser you can start the applications and modules that come with Ortho System by selecting the desired model set and clicking the corresponding icon:

  • - to start Preparation (Custom base workflow)
  • - to start Clear Aligner model (automatic)
  • - to start Clear Aligners

  • - to start Horseshoe model (automatic)

  • - to start Retainer model (automatic)
  • - to start Bracket Placement
  • - to start Ortho Analyzer
  • - to start Appliance Designer
  •   - to access Exports
