The Input options tab contains functionality relevant to accessing the patients' database and loading a head scan into an order accordingly.

 

The Patient browser option opens the patient browser with patients' database, allowing you to select the desired item to work with.

 

The Open DICOM model option allows you to load the volumetric data (head scan) coming from, e.g., Cone Beam CT scanners for better analysis and alignment during the treatment process.

 

Click Open DICOM model  in the Input options tab of Ortho Analyzer to load the CBCT/CT scan into the patient case. Select the desired file and click Next. The DICOM loader window opens:

 

 

In the views section (1) you can move the control squares for a better scan overview. You need to mind the key parameters that will influence your further work with the DICOM scan: Memory available (2) and Memory required (4). If the required memory exceeds the available memory, it is possible to reduce the scan size by using the Cropping tool (3). Click Next to proceed.

 

 

Alignment with CT Scan allows you to align the model with the loaded CBCT/CT scan.

 

Note: The CT scan is only valid for the alignment step. It cannot be taken as the actual occlusion you will be working with in Ortho Analyzer.

 

Click Alignment with CT Scan in the Model Alignment tab to start the process of alignment.

 

Note: The Alignment with CT Scan option is unavailable if the CBCT/CT scan has not been successfully loaded. To activate this option, you need to load a CBCT/CT scan first.

 

Click Dual View if you want to activate the dual window view for an easier alignment process. Use the CT  visualization slider to visualize the CT scan. Place three points according to the scheme to the left of the window. For better alignment, you can adjust the visibility of the objects with the relevant visualization sliders in the upper right corner of the window:

 

Slider

Description

Maxilla - allows you to show/hide as well as adjust the transparency level for the maxillary model.

Mandibular - allows you to show/hide as well as adjust the transparency level for the mandibular model.

CT - allows you to adjust the visualization of the CT scan.

Skin alpha value - allows you to customize the way the skin is displayed.

Bones alpha value - allows you to adjust the visualization of the bones of the CT scan.

Enamel alpha value - controls the way the enamel is visualized.

 

Click Align to finish the procedure. Once alignment is done, you can switch the CT scan to fit to lower or upper model with the help of the corresponding visualization slider:

 

Slider

Description

Switch to fit to lower/upper model - allows you to switch between fitting the CT scan to the upper or lower model.
