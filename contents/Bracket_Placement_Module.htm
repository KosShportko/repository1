The Bracket Placement option in the function panel allows you to accurately position brackets used for straight wire technique on the patient’s malocclusion in order to optimize the treatment efficiency and reduce bonding chair time. The brackets positions can then be transferred to Appliance Designer where the transfer media can easily be designed with the dedicated bracket transfer. To place the brackets, go through the workflow described below.

 

Note: The Bracket Placement option is an add-on to Ortho Planner and requires a specific license activation. Please, contact your 3Shape reseller to activate additional modules.

 

Note: Make sure that the arch shapes and bracket libraries have been downloaded via the download center on the Settings page by going to Settings > Download center before you can use this feature.

 

Note: You can also open Bracket Placement from the patient browser by selecting the required model set and clicking the Bracket Placement icon.

 

Using the Auto complete option allows to start the IDB workflow with the final bracket placement.

 

Note: The Validate button will be disabled until you revisit the Arch shape placement and Bracket adjustment steps.

 

The Bracket Placement workflow includes the preparation steps, where planes are set and segmentation is done.

After this, you can adjust the position of the FA points on each tooth at the Rotation center substep. The default position of an FA point is defined at the intersection of the projected long axis on the vestibular side of the tooth and the mid-transverse plane of the crown. The mid-transverse plane of the crown is calculated from the top of the tooth at the gingival contour to the tip of the tooth.

 

 

Perform the arch shape placement on Mandibular and Maxillary models.

 

  •  Dual view mode - splits the window in two, showing the malocclusion on the left side and the resulting setup on the right side of the window simultaneously.
  •  Left side only - shows only the malocclusion.
  • Both sides on one window - shows both views - the malocclusion and the resulting setup on the same window.
  •  Right side only - shows only the resulting setup.

 

Note: You can check the available arch shapes and create custom ones on the Settings page by going to Settings > Bracket placement > Arch shapes.

 

Perform the bracket placement on Maxillary and Mandibular models.

 

Note: You can access available bracket libraries on the Settings page by going to Settings > Bracket placement > Bracket libraries.

 

Option

Description
Force tip and/or inclination Forces the tip and/or inclination indicated in the chosen prescription for bracket placement while disabling the auto-snap option for selected teeth.
Use distance to gingival contour Considers the distance to gingival contour indicated in the chosen prescription for bracket placement instead of distance to incisal edge.
Apply Applies the selected prescription to the selected teeth.
Apply to all Applies the selected prescription to all teeth.

 

Note: To be able to use Bracket prescriptions, you must first create and set them ion the Settings page by going to Settings > Bracket placement > Bracket prescriptions.

 

Perform Bracket movements by selecting the needed movement option and moving the bracket on each particular tooth:

  •  Move the bracket on the tooth surface without constraints.
  •  Move the bracket on the tooth surface along the horizontal axis.
  •  Move the bracket on the tooth surface along the vertical axis.

 

Click the Distribute teeth evenly button to automatically adjust the resulting setup, visualized to the right when in Dual view mode  or in Both sides on one window , so that all teeth are arranged evenly along the arch.

 

Note: In case missing tooth width is set within the Segmentation step in Bracket Placement workflow, the Distribute teeth evenly feature takes the missing tooth into account when re-arranging the teeth.

 

 

You can adjust movement increments in the Movement distance box and indicate the maximum allowed distance from the bracket to the tooth in the Max bracket-tooth distance box.

 

Option

Description
Disable collision detection to antagonist

Allows to position the bracket by disregarding the constraints from the antagonist, however applying collision detection to the considered tooth surface, and to optimize the bracket position on the lower jaw despite occlusal interferences with the upper jaw. Using the virtual articulator then allows you to open the bite of the patient to avoid these occlusal interferences.
Disable collision detection to gingiva

Allows to position the bracket by disregarding the constraints from the gingiva, however applying collision detection to the considered tooth surface.

Hide collisions of brackets

Hides collisions, which are marked in red, between the brackets and with the antagonist.

Move teeth symmetrically

Allows you to simultaneously adjust the position of two symmetrical teeth by only moving one of them. Both symmetrical teeth are highlighted. The movements on one tooth are mirrored by the other, thus creating symmetrical adjustments. For this, you can move both a bracket on the malocclusion or a tooth itself on the resulted setup, or use the arrow keys on your keyboard.

Allow wire bending

Highlights in red the place where the arch wire must be bent due to tooth movements.

Disable auto-snap

Allows to adjust the position of the bracket manually in all dimensions using the control points. The bracket movement from the tooth using the blue control point is constrained by the value in the Max bracket-tooth distance box.

 

Auto-snap is the option that automatically ensures maximum contact of the bracket on the tooth surface. If it doesn’t perform properly initially, disabling this option while manually positioning the bracket and then enabling it again can usually solve the issue. When auto-snap is disabled, you can click Force tip to apply the tip and torque prescriptions from the bracket library manufacturer.

 

Hint: Use visualization sliders in the upper right corner of the window to enable/disable and adjust visibility of specific elements during bracket placement.

 

The Difference from malocclusion table contains Bracket and Wire Bending parameters and their values which represent the difference between the malocclusion and resulted setup for a selected tooth. Those parameters are defined below:

 

Parameter Description
Inclination Defines the inclination value in vestibular (+) or lingual (-) direction.
Tip Defines the angulation value in mesial (+) or distal (-) direction.
Rotation Defines the rotation value in mesial (+) or distal (-) direction.
In/Out Defines movements perpendicular to the arch shape itself. Where the bracket movements are no longer possible, the corresponding wire bending value comes into effect to accommodate for further tooth movements in vestibular (-) or lingual (+) direction.
Intrusions/Extrusions Defines movements perpendicular to the arch shape plane - for intrusion (-) or extrusion (+).
Sliding along the wire Defines the movement value along the arch wire.
Incisal Edge Distance Defines the distance from incisal edge center to a tooth.
Gingiva Distance Defines the distance from gingiva contour to a tooth.
Tooth long axis angle Displays the angle of the tooth long axis.
Teeth movements overview Opens the corresponding table with values representing difference from malocclusion for each tooth in the arch.
Rotation interval Controls the value by which angular parameters (e.g. inclination, tip, rotation) are changed when you adjust them using small arrow buttons.
Translation interval Controls the value by which distance parameters (e.g. in/out, intrusion/extrusion, sliding along wire, incisal edge distance, gingiva distance) are changed when you adjust them using small arrow buttons.

 

The positioning parameters are defined below:

 

Parameter

Description
Tip

The angulation of the long axis of tooth (or bracket) to the reference plane in the mesial distal direction.
Torque*/Inclination

While the torque is the built-in bracket property - the angle between the bracket base, above the FA point, and the wire slot bottom - the inclination is the resulting angle of the tooth that the built-in torque induces in relation to the arch wire plane.
In/Out

Distance from the inner side of the bracket slot to the tooth surface at the entrance of the wire (mesial side)/Distance from the outer side of bracket to the tooth surface at the exit of the wire (distal side).

Incisal Edge Distance

Distance to the incisal edge from the FA point or bracket base point.

Gingiva distance

Distance from the gingival contour to the FA point or bracket base point.

Mesial Wire Distance

Distance from the arch wire to the mesial side of the tooth.

Distal Wire Distance

Distance from the arch wire to the distal side of the tooth.

Base Point Distance

Distance from the base point at the center of the bracket to the tooth surface.

Max Tooth Distance

Maximal distance from the bracket to the tooth surface.

 

Check all the settings for the brackets and validate their positioning with the validation tools so the bracket transfers can be designed in the Bracket Transfer module of Appliance Designer. Validation provides a reliable exchange and review process of indirect bonding cases between locations (e.g., partner clinic, lab).

 

Note: Validating the model set will save with the brackets positioned in the way you did in Ortho Analyzer ready for the transfer to Appliance Designer.
