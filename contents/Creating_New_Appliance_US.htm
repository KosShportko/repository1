In the patient browser, select a model set, click , and click to open the Add Appliance window. In the Add Appliance window, enter Appliance ID, choose the appliance type you want to produce from the Appliance type dropdown list, and add other relevant information. When done, click .

 

The function panel in the upper left corner of the window allows you to create components of your appliance and add features, which is allowed in particular steps of the workflow.

 

Shell

A Shell is a surface wrapped around the surface of the source model, which thickness and offset distance are defined.

 

Note: The name of the component corresponds to the name of the Output model.

 

Note: Appliance Designer allows you to build any component on either the tooth model, or on any other component created previously.

 

Insertion direction

 

Click the Set insertion direction button to set insertion direction from your own view. It is possible to reset the insertion direction to default view (1) or copy the current insertion direction to default view (2). The Show colors checkbox activates the Undercut depth color map. When done, click Next to proceed to the next step.

Bar

A Bar is a 3D model based on a 2D profile that allows to apply predefined parameters and geometries.

 

Note: To be able to use a Bar component, you need to define the bar 2D profiles on the Settings page by going to Settings > Appliances > Bars.

Create Bar

 

When moving to the Create Bar step, the bar on the model is created automatically. Select the bar type from the corresponding drop-down menu.

 

Note: You can add/edit the bar profiles on the Settings page by going to Settings > Appliances > Bars.

Combine models

The Combine models feature allows you adding or subtracting two models geometries.

 

Note: The model you want to subtract from should be selected first from the list of the models.

 

In the function panel, click the Add Combine models step button and set properties of the Combine models component:

  • Combine - allows you to combine geometries of two models into one new solid model.
  • Subtract - will carve the geometry of one model according to the other selected model.
  • Extra Combine - allows to combine two models while carving the underlying dentition away automatically.

 

For higher-quality results, select the Check and Repair models checkbox.

 

Remove undercuts

The Remove undercuts feature allows to fill undercuts to simulate e.g. waxing on models. The output of this process is a modified master model ready for further design.

 

Remove undercuts

 

Click the Set insertion direction button to indicate your own insertion direction. It is possible to reset the insertion direction to default view (1) or copy current insertion direction to default view (2). Show colors - indicates the areas on the model according to the Undercut depth color map. You can observe the areas that will be removed on the model.

 

 

Wax trimming

 

Use the Wax knife settings (similar to the Sculpt tool) to perform a manual wax trimming of the desired areas to create local retention on the master model.

 

Model hollowing

The Add Model Hollowing step option allows to hollow out a 3D model in order to save material for rapid prototyping.

 

In the function panel, click the Add Model Hollowing step button and set properties of the Model hollowing component.

 

 

Bracket transfer master model

The Bracket transfer master model option allows you to create a model for either thermoforming (master model with markers or containers) or continuous digital printing of directly printed transfer trays.

 

In the function panel, click the Add Bracket transfer master model step button.

 

If you use a printer with short-term biocompatible materials, it is possible to design and print a transfer tray based on the master model with containers and hence skip the thermoforming.

 

Once you get to the Shell step, indicate the insertion direction and create a spline that will define the shell. Based on the rigidity of the material, it is possible to make ridges in the design to increase flexibility. In the function panel, click the Add Sculpt button and then click the Modify model step.
