To start Appliance Designer, in the patient browser, select a model set, click , and either click to create a new appliance design or click to open an existing one. The Appliance Designer software allows you to produce appliances consisting of parts and components which are used for orthodontic treatment.

 

The Appliance Designer user interface looks as follows:

 

 

(1)

Function panel (incl. tabs for components and features)

(2)

Visualization toolbar

(3)

Visualization sliders

 

Clicking the 3Shape logo in the bottom right corner of the session window or using the F1 shortcut on your keyboard opens the window with the information about the software and your subscriptions, also providing you access to online support, training center, user manual, 3rd-party licenses, and submitting a feedback report.
