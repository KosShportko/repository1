You can open the Cross section function from the Visualization toolbar to access all 2D-based planes, measurement and symmetry tools. This feature also lets you upload and overlay 2D images on any cross-section plan. You can use 2D overlay images to take any 2D measurements on the image and grid.

 

 

(1) Measurement buttons

(2) Cross-section plane

(3) 2D Cross-section preview

(4) Manipulation buttons

 

View options - includes the buttons that set the viewing parameters:

 

Button Description

Clipping plane - Left  - hides the right side of the model in relation to the cross-section plane.

Clipping plane - Right - hides the left side of the model in relation to the cross-section plane.

Fixed camera - locks the plane in one position and moves the model instead (only applicable to the Plane offset slider).

Grid - turns on/off the grid on the plane.

Clip planes lines by grid - clips plane lines outside the grid on the 3D overview.

 

2D overlay - allows you to add a 2D image to the patient case:

 

Button Description

New 2D Overlay - allows you to load a new image in PNG, JPG or BMP formats.

Clear 2D Overlay - removes the current 2D overlay image.

Modify 2D Overlay - allows you to modify the 2D overlay image: scale, rotation and position, using the dedicated control points.

Mirror 2D Overlay Image - flips/mirrors the 2D overlay image.

 

2D Cross-section control buttons

 

Button Description

Along Dental Arc - allows you to position the cross-section along the dental arch with the spline going through its center.

Standard planes - allows you to select the plane from the list of the planes you create.

Save plane - allows you to save the created plane.

Clear View - removes the cross-section.

Measure area - allows you to measure a particular area on the model in the 2D Cross section window.

Clear measure - deletes the measures.

Zoom all - allows you to zoom the model.
