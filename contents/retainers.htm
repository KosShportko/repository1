Horseshoe model (automatic) and Retainer model (automatic)   workflows provide functionality for quick and effective retainer production.

 

 

Note: Horseshoe/Retainer model (automatic) workflows can only be accessed when the user is logged in into 3Shape Communicate. In case you have not set up your 3Shape Communicate account yet, you will be guided to proceed on Communication page in Settings.

 

 

This workflow is not intended to make models that are going to be used in the Clear Aligner workflow, as hollowing and ID tags could potentially be applied twice on the same models. If you intent to make models used for the Clear Aligner workflow, please use the Clear Aligner model (automatic) workflow.

 

The Horseshoe model (automatic) and Retainer model (automatic) workflows provides you with the following tools:

 

Automatic jaw base - allows you to apply automatic base to the jaw according to the default parameters. You can change the values of Base safety margin size for maxilla and mandible using the corresponding sliders or inserting the desired values.

 

 

Adjust your result using the tools available at the Sculpt Maxillary and Mandibular  steps.

 

The Wax knife tool allows you to add/remove/smooth material in the selected areas.

 

The Morphing tool allows you to stretch the target area inwards and outwards.

 

The Plane cut tool allows you to cut off parts of the model using a plane. Draw a line in the desired place to create a plane. You can move and tilt the plane using the control points. To change the direction in which the plane is applied, click the Reverse cut direction button.

 

Increase contrast option allows you to disable colors of the model and switch to high-contrast material to be able to assess any problematic areas on the Sculpt steps of the workflow.

 

Hollowing - allows you to hollow out the inside of the model to save material consumption while retaining the needed structural stability. Before hollowing, you can choose a machine and material, and the software will suggest the recommended wall thickness values. You can also insert the drain holes on the models for better print quality and easier post processing. You can adjust the radius of the drain holes, as well as their position and number.

 

Fixture application tool – allows you to place fixtures at the bottom side of the upper and lower jaw models.

 

Note: The Fixture option is an add-on to Ortho Planner and requires a specific license activation. Please, contact your 3Shape reseller to activate additional modules.

 

Cut line - allows you to manually edit a cut-line per each jaw. Click and draw the part of the cut-line that requires correction and the line will be adjusted. You can set such parameters as Offset, Wave offset and Retainer termination type. The cut-line output is generated in .dxf and .pts formats.

 

ID Tags - allows you to add identification on the surface of the model. Input text and/or choose an attribute from the available list. Move the tag on the models using the arrow buttons.

 

Marking Tag Simulation - allows you to export the position of the tag for laser engraving. As a result of export, a SVG file is generated with necessary information for engraving the tag on the model, along with the file containing the jaw model.

 

Graphics Marking Tag Simulation - allows to identify an area for boundaries that could contain the graphics tag. After the export, the information on the graphics marking tag is written to a SVG file along with the corresponding information on the marking tag.

 

Note: In order to use Marking Tag Simulation, Graphics Marking Tag Simulation, Fixture application tool and Cut-line1 options, ensure that the appropriate licenses are enabled on your dongle. For more details, contact your reseller.

On the last step of the workflow click Save to export the model set, or Done to save the changes and exit Retainer window.
