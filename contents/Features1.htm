The features are the specific design parameters of the components. But there are also additional features that can be applied to an existing component.

 

The features tab is automatically opened if you have selected the Start now checkbox while creating the new component.

 

It is possible to use the Current workflow wizard operation buttons to customize your steps:

 

Button Description

Previous - allows you to undo the changes.

Next - allows you to move to the next step.

Done - finalizes the customization process.

Add Feature - opens the drop-down menu with additional features.

 

You can edit the properties of the Create shell and Bar creation postprocessing steps in the features tab by right-clicking the appropriate step and choosing Set step properties. In the appeared Step properties window, deselect the options you don't want to use in this step and click OK. This will allow to hide some specific component operations from your workflow. For example, if you use a global remove undercuts feature for your modeling process, you are able to remove the similar feature from the bar or shell component. It is possible to add hint texts and images to all available features.

 

Note: Hint texts and images can be added to any selected feature in order to give more details on how to apply the feature to a modeling template (e.g., size of tools, position of control points, etc.).
