A brief overview of the function panel is presented below.

 

The Input options tab includes the following options (see the chapter Input and Alignment of CBCT/CT Scans):

 

Option

Description

 Patient browser - opens the patient browser with a patients' database.

Open DICOM model - allows you to load the CBCT/CT scan of the head into the order.

 

The Model Alignment tab includes the following alignment options:

 

Option

Description

Alignment with CT Scan - loads the window for performing the alignment with CBCT/CT scan procedure (see the chapter Input and Alignment of CBCT/CT Scans).

Occlusion - helps realign the upper and lower virtual casts (see the chapter Occlusion).

Alignment for open model set - helps align the upper and lower virtual casts for the open model to make the visualization and manipulation of the model more convenient (see the chapter Alignment for Open Model Set).

 

The Preparation tab assists with indicating landmarks on the virtual model set and includes the following options (see the chapter Preparation):

 

Option

Description

Setup occlusion plane - lets you set up the occlusion plane of the model set.

Standard Planes - allows you to specify the point on the model which will subsequently serve as local orientation point from where the coordinate system axes originate.

Segmentation of maxillary model - starts the definition and segmentation of individual teeth on the maxillary side.

Segmentation of mandibular model - starts the definition and segmentation of individual teeth on the mandibular side.

 

The Analysis tab options are sorted according to the previously executed options; they typically rely on the selections and definitions made in the preceding tabs. The tab includes the following options (see the chapter Analysis):

 

Option Description

Questionnaires - allows you to present a customized analysis.

Overbite/Overjet - provides overbite/overjet measurements for easy diagnostics.

Dental arch analysis - provides calculations of the arch lengths based on the selections made in the Preparation tab and enables the creation of an ideal arc shape to be used for the treatment.

Analysis reports - provides detailed analyses of various orthodontic data for both maxillary and mandibular casts, comparing them with the standard tables preset in Settings. This involves inspecting the tooth widths, space, anterior and Bolton ratios.

Generate Report - creates a ready to print analysis report based on the analysis data.

Occlusion map - visualizes the occlusal distances in the model set according to the color scale applied to the virtual casts.

Compare model sets - enables immediate comparison of model sets visualizing the two simultaneously.

 

The Treatment tab includes the following treatment options:

 

Option

Description

Virtual Setup - starts treatment simulation via virtual setups (see the chapter Treatment).

Bracket Placement - opens the Bracket Placement module for brackets positioning (see the chapter Indirect Bonding).

Clear Aligners - starts the workflow to plan treatment for further production of aligners (see the chapter Clear Aligners).

 

The Custom Workflows tab includes the following predefined workflow options (see the chapter Custom Workflows):

 

 Option

Description

Analyzer - starts the workflow for a typical model set analysis.

Preparation - starts the workflow to perform a model set preparation.
