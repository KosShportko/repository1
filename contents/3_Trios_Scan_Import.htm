Import TRIOS scans

Ortho System works with TRIOS scans as well as third-party model sets.

 

There are four ways to import the TRIOS scans into Ortho System:

  • Receiving orders via TRIOS Direct Connection is the method to transfer TRIOS scans locally: it only works if TRIOS and Ortho System are installed on the same Local Area Network.
  • Sending/receiving orders via 3Shape Communicate allows to send scanned file directly over the Internet, using 3Shape’s cloud based solution called 3Shape Communicate.
  • Right-clicking a patient in the patient browser and selecting Import case(s)/patient(s) from file in the contextual menu.
  • Dragging-and-dropping the .3sz and .3oxz files directly into the patient browser.

 

Both TRIOS Direct Connection and 3Shape Communicate are set up on the Settings page (see the chapter Settings). Refer to the dedicated technical guide to set up the TRIOS unit properly for these file exchange methods.
