Clear Aligner model (automatic) workflow allows you to apply automatic base to the jaws. After selecting an unprepared model in the patient browser, click the Preparation -> Clear Aligner model (automatic) to start the workflow.   

 

Note: Clear Aligner model (automatic) workflow can only be accessed when the user is logged in into 3Shape Communicate. In case you have not set up your 3Shape Communicate account yet, you will be guided to proceed on Communication page in Settings.

 

These models are only intended for clear aligner cases, where the potential ID tag and hollowing will be applied later in the Сlear Аligner workflow. If you intent to have a model with ID tag and hollowing generated, that is not for clear aligners, please use either the Horseshoe model (automatic) or Retainer model (automatic) workflows.

In the Automatic jaw base step, the base is automatically applied to the jaws according to the default parameters. You can change the values of Base safety margin size for maxilla and mandible using the corresponding sliders or inserting the desired values.

 

 

You can use the Undo / Redo options to cancel or restore the latest or previous operations to the automatic base application parameters.

 

When the base is created, click Next to proceed to the next step. Adjust your result using the tools available at the Sculpt Maxillary and Mandibular steps.

 

The Wax knife tool allows you to add/remove/smooth material in the selected areas.

 

The Morphing tool allows you to stretch the target area inwards and outwards.

 

The Plane cut  tool allows you to cut off parts of the model using a plane. Draw a line in the desired place to create a plane. You can move and tilt the plane using the control points. To change the direction in which the plane is applied, click the Reverse cut direction button.

 

Increase contrast option allows you to disable colors of the model and switch to high-contrast material to be able to assess any problematic areas on the Sculpt steps of the workflow.

The Fixture tool allows to place fixtures at the bottom side of the upper and lower  jaw models.

 

Note: The Fixture option is an add-on to Ortho Planner and requires a specific license activation. Please, contact your 3Shape reseller to activate additional modules.

 

On the last step of the workflow click Save to export the model set, or Done to save the changes and exit Clear Aligner model window.
